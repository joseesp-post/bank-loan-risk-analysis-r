# Bank Loan Risk Analysis using R

This project analyzes bank loan data to identify factors associated with loan approval and credit risk.

The objective is to explore the relationship between customer financial characteristics and the likelihood of receiving a loan.

## Project Objectives

- Analyze customer financial data
- Identify variables associated with loan approval
- Explore potential patterns in credit risk

## Methodology

The analysis was conducted using **R in RStudio**, including:

1. Data exploration
2. Data cleaning
3. Statistical analysis
4. Data visualization
5. Interpretation of results

## Key Steps

- Import and preprocessing of the dataset
- Exploratory Data Analysis (EDA)
- Correlation analysis
- Visualization of key financial variables
- Interpretation of insights related to loan approval

## Tools Used

- R
- RStudio
- Data visualization libraries
- Statistical analysis methods
